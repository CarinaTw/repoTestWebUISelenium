Скрипт собирает статистику серверного лога.
- Общее количество запросов
- Количество GET / POST запросов
- ТОП 10 ip-адресов, с которых были отправлены запросы
- ТОП 10 самых долгих по времени выполнения запросов
- Количество клиентских ошибок по статус кодам и информацию по каждому запросу (status_code, ip, method, url)
- Количество ошибок сервера по статус кодам и информацию по каждому запросу (status_code, ip, method, url)

Для запуска скрипта:
1) перейти в директорию
/repoTestWebUISelenium/parser
2) запустить скрипт командой
python log_parser.py --filename_original "access.log" --filename_result "result.json"
